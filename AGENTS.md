# AGENTS.md

## Project

Music Auto Show is a Rust service with an embedded Vite SPA. Rust captures and analyzes audio, runs native BeatNet+ inference, ports the original lighting algorithms, drives DMX, and exposes protobuf services through tonic-web. The React SPA consumes that API with Buf-generated types and Connect gRPC-Web.

The repository must remain Rust and TypeScript only. Do not reintroduce Python, NiceGUI, madmom, PyInstaller, Conda, or pip-based runtime paths.

## Git

- Use Conventional Commit messages in the form `<type>(<scope>): <subject>`.
- Prefer `feat`, `fix`, `docs`, `refactor`, `test`, `chore`, `ci`, `build`, and `perf`.
- Keep the subject imperative and concise.
- Do not create a branch unless the user explicitly asks.
- Never bypass commit hooks or Lefthook. Do not use `-n`.
- Preserve unrelated worktree changes.

## Commands

```bash
# Install dependencies
bun install --frozen-lockfile
bun install --cwd frontend --frozen-lockfile

# Run with simulated audio and DMX
cargo run -- --simulate

# Run the frontend dev server against Rust on port 3000
bun run frontend:dev

# Full validation
bun run check

# Release build with the SPA embedded
bun run build
```

Targeted checks:

```bash
bun run proto:format
bun run proto:lint
bun run proto:generate
bun run --cwd frontend format:check
bun run --cwd frontend typecheck
bun run --cwd frontend lint
bun run --cwd frontend test
cargo fmt --all -- --check
cargo test --all-targets
```

## Architecture

- `proto/music_auto_show/v1/music_auto_show.proto` is the shared API contract.
- `src/audio.rs` owns capture, analysis, resampling, and recording.
- `src/beatnet.rs` owns BeatNet+ features, inference, and causal decoding.
- `src/effects.rs` owns visualization modes, movement modes, fixture mapping, smoothing, and universe output.
- `src/dmx.rs` owns Open DMX and simulated output.
- `src/config.rs` owns defaults, built-in profiles, JSON persistence, and legacy migration.
- `src/app.rs` owns the runtime lifecycle and snapshot stream.
- `src/api.rs` implements the tonic service.
- `frontend/src/lib/show-api.ts` is the Effect service boundary around Connect.
- `frontend/src/routes` contains TanStack Router file routes.

The Vite build is embedded with `rust-embed`. The SPA and gRPC-Web API must continue to work on one production port.

## Rust style

- Use idiomatic Rust 2024 and keep `cargo fmt` clean.
- Represent recoverable runtime failures with `Result` and useful context.
- Clamp all DMX channel output to both the channel range and `0..=255`.
- Keep the real-time path bounded. Avoid blocking work while holding async locks.
- Preserve graceful fallback when audio, media, BeatNet+, or DMX hardware is unavailable.
- Add focused unit tests when changing analysis, decoding, configuration migration, fixture mapping, or DMX behavior.

## Frontend style

- React components use `PascalCase`. Hooks and utilities use `kebab-case` filenames.
- Use TanStack Router, Query, Form, and Table v9 through their current APIs.
- Put asynchronous API behavior behind the Effect `ShowApi` service.
- Use Credenza for every modal flow. Do not consume dialog primitives directly in feature code.
- Style application code with Tailwind utilities and the preset's CSS variables. Do not use raw DOM `style` props for UI styling.
- Canvas colors and coordinates may be calculated directly when they represent live visualization data.
- Use `gap-*`, never `space-*`.
- Never use an array index as a React key. Create or use stable domain identities.
- Use `generateN` for numbered skeleton placeholders.
- Prefer consumer-side fixes before changing shared primitives.

## Frozen and generated paths

- Never edit `frontend/src/components/ui/**`. These files must stay exactly as generated by the shadcn preset.
- Oxfmt and Oxlint intentionally exclude `frontend/src/components/ui/**`.
- Do not hand-edit `frontend/src/gen/**` or `frontend/src/routeTree.gen.ts`.
- Change protobuf definitions, then run `bun run proto:generate`.
- Let the TanStack Router plugin regenerate the route tree.

## BeatNet+

- Keep inference native. Do not shell out to Python.
- Preserve the official 22,050 Hz, 1,764-sample window, 441-sample hop, 288-feature, convolution, projection, four-layer LSTM, and three-class topology.
- Do not commit checkpoint files. `models/*.pt` and `models/*.pth` are user-supplied because upstream redistribution terms are not provided.
- Degrade visibly when the checkpoint is missing or invalid. The show must still run.

## Effect source reference

Run `bun run prepare` when local Effect source is needed. It clones the reference checkout into `.repos/effect`, which is ignored and must not be committed.
